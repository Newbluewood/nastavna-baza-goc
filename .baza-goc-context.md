# Baza Goč - Project Context (Presek Projekta)

**Verzija:** 16. april 2026. (Nakon Faze "Re-dizajn Smeštaja")
**Svrha:** Persistentan referentni fajl (Memory) koji opisuje trenutno stanje aplikacije, tehnički stog (stack) i buduće pravce razvitka.

---

## 1. Sistemska Arhitektura
* **Tip aplikacije:** Full-stack CMS + Javni Portal (Mesto: `c:\Users\Korisnik\Desktop\Antigravity\baza-goc-priprema`)
* **Backend:** Node.js (Express), `index.js` (rutiranje), `db.js` (konekcijski pul). Hostovan na Render platformi.
* **Baza Podataka:** MySQL na Aiven klaudu (URL i port definisani kroz `.env`).
* **Frontend:** Vue 3, Vite, Vue Router, Pinia (`langStore`). Autentifikacija je JWT token (7 dana). Planiran host: Netlify.
* **Prevođenje:** DeepL API. Koncept "Blizanačkog unosa" (svaki unos u bazi ima bazni (SR) i `_translations` produžetak za EN jezik).

## 2. Implementirane Funkcionalnosti (Trenutno Stanje)
- **Vesti (News):** 
  - Kreiranje iz Admin Panela uz automatski DeepL prevod.
  - Automatsko generisanje *Slug*-ova za UI navigaciju (npr. `/vesti/nadstresnica`). Podržava i ID i slug dohvatanje.
  - Gosti mogu ostaviti *Like* (Praćeno kroz LocalStorage prevenciju duplih lajkova).
- **Početna Strana (HomeView):** 
  - Centralizovan `PageTemplate` za iscrtavanje Hero slajdera (iz `hero_slides`), vesti u Carousell obliku, globalni tekst, opšta galerija.
- **Smeštaji (SmestajView):** 
  - Custom grid prikaz svih "facilities" type 'smestaj'.
  - Učitane komplementarne galerije (Lightbox integracija) + `floor_plan_image` (Skica sobe).
  - Integrisan Google Maps Iframe na dnu.
- **Upiti (InquiryModal):** 
  - Popup prozor iz kog korisnik šalje poruku za *selektovanu sobu*, predativši ime, email, telefon i poruku direktno u MySQL bazu (`inquiries`).

## 3. Plan za Narednu Fazu (Booking.com Model)
* **Sadržaji Soba (Amenities):** Potrebno refaktorisati bazu da svaka soba ima specificirane amenities zastavice (WiFi, TV, Kuhinja, Parking itd.) i prikazati ih kroz univerzalne ikone na frontendu.
* **Bedževi za Lokaciju:** "20 min od centra kolima", "5 min do žičare pešaka".
* **Sistem Rezervacija & Kalendar:** 
  - Zamena klasičnog tekstualnog unosa datuma pravim interaktivnim kalendarčićem (Vue Datepicker).
  - Kreiranje `reservations` tabele gde će Admini unositi/odobravati kalendarske popune.
  - Zaključavanje UI kalendara: Gostima onemogućiti klik na datume koji su zauzeti.

## 4. Baza Podataka (Ključne Tabele)
1. `admins` - JWT kredencijali.
2. `pages` / `page_translations` - Tekstualni statični sadržaj (npr. Početna, Smeštaj).
3. `hero_slides` / `hero_slides_translations` - Hero slike sa naslovima i subtitle tekstom.
4. `facilities` / `facility_translations` - Prikaz soba u Smeštaju. Ima bitne kolone: `capacity`, `cover_image`, `floor_plan_image`.
5. `news` / `news_translations` - Vesti. Generiše unikatan `slug`. Brojač `likes`.
6. `media_gallery` - Skupne slike; veze se preko polja `entity_type` ('facility', 'news', 'page') i `entity_id`.
7. `inquiries` - Upiti korisnika za smeštaj.

## 5. Važni Podsetnici
* **Aesthetics Matter:** Zadržati strogi `border-radius: 0;` (oštar dizajn) i uvek pratiti boje iz `--color-nav` itd.
* **Prevođenje:** U View template-ima striktno obezbediti dinamički redner koristeći `langStore.currentLang === 'sr' ? 'Tekst' : 'Text'`.
* **Deployment Workflow:** Svaka modifikacija zahteva brzi komit (`git push`) jer Aiven oblači bazu uživo, a Netlify/Render povlače kod sa GitHub repozitorijuma.
