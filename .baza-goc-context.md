## 1. Sistemska Arhitektura

- **Lokacija projekta:** `c:\Users\Korisnik\Desktop\Antigravity\baza-goc-priprema`

- **Hosting:** Backend → **Render**, Frontend → **Netlify** (`nastavnabazagoc.netlify.app`)
- **Prevođenje:** DeepL API (sa mock fallback ako key nije podešen)

## 3. Implementirane Funkcionalnosti

### 3.1 Javni Portal

- **Smештај (`/smestaj`):**
  - Grid prikaz zgrada (facilities type='smestaj')
  - Svaka kartica ima: sliku, `location_badges` (JSON → bedževi), kapacitet, opis
  - `📍 Otvori Mapu` link → `maps.google.com/?q=lat,lng` (mobilni otvara GMaps app)
  - `🏠 Pogledaj Sobe` vodi na `/smestaj/:id`

- **Sobe (`/smestaj/:id`):**
  - Booking.com stil: slika levo, info desno
  - Amenities bedževi (JSON kolona → mapiranje u SR/EN label + emoji ikonu)
  - Dugme "Pošalji Upit" otvara `InquiryModal`

- **InquiryModal:**
  - VueDatePicker kalendar sa range selekcijom
  - `disabledDates` se fetchuju sa `/api/rooms/:id/availability`
  - **Fix timezone bug:** datumi se parsiraju sa `T12:00:00` sufixom da ne dođe do pomaka
  - Zaglavlje jasno pokazuje zgradu + sobu (sprečava zabunu)
  - Šalje na `POST /api/inquiries`

- **Hero Slider (globalno):**
  - Na mobilnom (< 640px) naslov se **centrira** `translateX(-50%)`
  - Implementirano u `PageTemplate.vue` → važi za SVE stranice
  - `SmestajView.vue` ima svoj isti fix u lokalnom `<style scoped>`

- **404 / Under Construction (`/404`):**
  - `NotFoundView.vue` → pametno detektuje poznate-ali-neimplementirane rute
  - Prikazuje 🚧 za "u izgradnji" ili 404 za nepostojeće rute
  - Catch-all ruta u routeru: `/:pathMatch(.*)*`

### 3.2 Admin Panel

- **Rute:** `/admin/login` → `/admin/vesti` → `/admin/rezervacije`
- **Svi admin routovi su zaštićeni** `authMiddleware` (JWT)
- **Sidebar:** Braon `#332317`, aktivni link `#cdac91`, hover efekt belo/braon

#### AdminNewsView (`/admin/vesti`)
- Tabela vesti sa braon headerom
- Forma za dodavanje vesti: SR originali + auto DeepL prevod na EN
- Dugme "+ Dodaj Vest" u braon stilu

#### AdminReservationsView (`/admin/rezervacije`)
- **Kartice filtera:** Svi / Novi / Odobreni / Odbijeni / Otkazani (sa brojevima)
- **Tabela:** Gost, Kontakt (email/tel kao linkovi), Objekat+Soba, Period, Status, Akcije
- **Akcije po statusu:**
  - Novo → `[✅ Odobri]` + `[❌ Odbiج]`
  - Odobri → automatski INSERT u `reservations` (zaključava datume u kalendaru!)
  - Odobreno → `[🚫 Otkaži]`
  - Otkaži → UPDATE `reservations.status = 'cancelled'` (oslobađa datume)

#### Admin Responsive (obe stranice)
- Na mobilnom (< 768px): sidebar skriven, pojavljuje se braon `☰ CMS Panel` dugme
- Klik otvara sidebar sa `transform: translateX(0)` animacijom
- Tamni overlay zatvara sidebar na klik

## 4. Navigacija / Router (`frontend/src/router/index.js`)

```
/                    → HomeView
/smestaj             → SmestajView
/smestaj/:id         → SmestajSingleView
/vesti/:id           → SingleNewsView
/admin/login         → AdminLoginView
/admin/vesti         → AdminNewsView (requiresAuth)
/admin/rezervacije   → AdminReservationsView (requiresAuth)
/:pathMatch(.*)*     → NotFoundView (catch-all 404)
```

## 5. Backend API Rute (`backend/index.js`)

| Method | Ruta | Opis |
|---|---|---|
| GET | `/api/home` | Početna: slides, page text, news |
| GET | `/api/smestaj` | Sve zgrade sa `latitude`, `longitude`, `location_badges` |
| GET | `/api/smestaj/:id` | Jedna zgrada + sobe sa `amenities` |
| GET | `/api/rooms/:id/availability` | Zauzeti datumi (DATE_FORMAT → čisti YYYY-MM-DD stringovi) |
| POST | `/api/inquiries` | Nova rezervacija/upit od gosta |
| POST | `/api/admin/login` | JWT login |
| POST | `/api/admin/translate` | DeepL prevod (protected) |
| POST | `/api/admin/news` | Dodaj vest (protected) |
| GET | `/api/admin/inquiries` | Svi upiti sa sobom i zgradom (protected) |
| POST | `/api/admin/inquiries/:id/status` | Promeni status upita + auto rezervacija (protected) |
| GET | `/api/news` | Lista vesti |
| GET | `/api/news/:id` | Jedna vest (po ID ili slugu) |
| POST | `/api/news/:id/like` | Lajkuj vest |

## 8. Dizajn Sistem

- **Font:** `FOF24` (Light 300, Medium 400, SemiBold 600, Bold 700) — lokalni `.otf` fajlovi
- **Paleta boja:**
  - `--c-braon-6: #332317` → Glavna tamna boja (nav, headeri, admin sidebar)
  - `--c-braon-5: #67462E` → `--color-nav`
  - `--c-braon-3: #CDAC91` → Akcenat, hover stanja, aktivni admin linkovi
  - `--c-braon-1: #FFF1E6` → Svetla pozadina
- **Stilski princip:** `border-radius: 0` svuda — oštar, inženjerski izgled
- **Nav separator `|`:** `opacity: 0.4`, negativni margin, skriven u mobilnom meniju

```bash

cd frontend && npm run dev

# 2. Reset baze (kada se menja šema)
cd backend && node setupDb.js

# 3. Pushovanje na produkciju
git add .
git commit -m "opis izmene"
git push
# → Render automatski redeploy backend
# → Netlify automatski rebuilduje frontend
```
