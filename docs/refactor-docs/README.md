# Refactor Docs Index

Ovaj folder je centralno mesto za planiranje, refaktorisanje i pripremu prezentacije.

## Dokumenti

1. REFACTORING_LOG.md
- Glavni dnevnik promena i status koraka.

2. AI_ROADMAP_AND_BUDGET.md
- AI strategija, graceful degradation, budzet i rollout faze.

3. PROJECT_CONTEXT_AND_SCOPE.md
- Opseg projekta, granice i prioriteti.

4. RELEASE_HANDBOOK.md
- Operativni koraci za release i provere pre deploy-a.

5. CLIENT_READINESS_CHECKLIST.md
- Checklist za klijentsku prezentaciju i produkcijsku spremnost.

6. MENTORSHIP_PLAN.md
- Plan rada po sesijama i način mentorisanog napretka.

7. SESSION_01_SQL_ALIGNMENT.md
- Beleške i zadaci za SQL usklađivanje.

8. SESSION_02_MIGRATION_SAFETY.md
- Beleške i zadaci za bezbednu migraciju.

9. learning-path-on-my-app.txt
- Lični plan učenja i reference na ključne dokumente.

## Pravilo rada
- Sve nove planske i kontekst dokumente dodavati u ovaj folder.
- Ako dokument referencira druge docs fajlove, koristiti putanje sa prefiksom docs/refactor-docs/.
- REFACTORING_LOG.md je obavezno mesto za upis značajnih promena.
