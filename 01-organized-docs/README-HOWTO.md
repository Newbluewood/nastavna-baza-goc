# HOWTO: Project Documentation

- All new info/context docs go in 01-organized-docs with a number prefix.
- Update 01-CHANGELOG.md for every significant change.
- Move old docs to 06-ARCHIVE if not current.
- Session/meeting notes go in 05-SESSION_LOGS.
- Backend/Frontend checklists in README-BACKEND.md and README-FRONTEND.md.
- For onboarding, start with 02-PROJECT_CONTEXT_AND_SCOPE.md.
