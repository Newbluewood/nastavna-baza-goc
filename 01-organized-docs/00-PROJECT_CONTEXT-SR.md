Нaставна база Гоч – Контекст пројекта

## Преглед пројекта
„Наставна база Гоч“ је веб платформа Шумарског факултета Универзитета у Београду. Омогућава динамичко управљање садржајем, резервације смештаја, CRM за госте и портал за посетиоце.

## Технолошки стек
- Фронтенд: Vue 3 (Composition API, <script setup>), Vue Router, Pinia
- Бекенд: Node.js, Express, MySQL (Aiven cloud), Nodemailer (Brevo SMTP преко API кључева)
- Стилови: Прилагођени CSS (модуларни, уграђен у Vue SFC), строг квадратни равни дизајн (без заобљених ивица), #332317 (навигација/текст) и #cdac91 (акценти)

## Архитектура
- Јавни портал: Почетна (/), Смештај (/smestaj), Вести (/vesti), Контакт/О нама (води на шаблоне „У изради“ преко NotFoundView.vue).
- Портал за госте: /prijava и /moj-nalog. Гости могу да виде резервације, откажу (до 7 дана пре доласка) и искористе ваучере лојалности у реалном времену.
- CMS / Админ: /admin/login, /admin/vesti, /admin/rezervacije, /admin/gosti. Обезбеђено преко JWT (admin_token).

## Шема базе података (кључне табеле)
1. pages: Динамички преводи страница (page_slug, title, text_content, lang).
2. hero_slides: Садржај за карусел по страницама (page_slug, image_url, target_link).
3. news: Динамички списак вести (id, title, content_text, image_url).
4. inquiries: Чува захтеве за резервацију. Статуси се проверавају у односу на доступност.
5. guests: (id, email, password_hash, name, phone, reservation_count, vouchers JSON, created_at). Праћење лојалности и ваучера.
